Trigonometry — Chapters 1 & 2 Equation Sheet
A personal reference sheet covering all core equations and identities from Chapters 1 and 2 of Trigonometry, built as part of both my Trigonometry and HTML Essentials coursework.

What's Inside

Angle conversions (degrees ↔ radians)
Arc length, sector area, and angular/linear speed formulas
Right-triangle trig ratios (SOH-CAH-TOA)
Unit circle key values table
Reciprocal, quotient, and Pythagorean identities — each with a worked example
Even/odd identities
Co-function identities
Quadrant sign rules (All Students Take Calculus)
Quick-reference summary

About This Project
This project was created for two courses:
Trigonometry — End of Chapter Assessment for Chapters 1 & 2. The goal was to build a well-organized, personalized equation sheet covering all identities and formulas, including worked examples for the reciprocal, quotient, and Pythagorean identities.
HTML Essentials — The sheet is hand-coded in a single HTML file using semantic HTML and CSS. It features a sticky navigation bar, color-coded sections, a responsive layout, and a reference table — all without any external frameworks.
Built With

HTML5
CSS3 (custom properties, grid, flexbox)
Google Fonts (DM Serif Display, DM Mono, Outfit)
